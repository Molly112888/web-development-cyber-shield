 Cyber Shield
Cyber Shield is an educational website designed to teach people about the threats they may encounter in the digital world. It provides structured courses, lessons, and quizzes on topics such as phishing, trojans, keyloggers, ransomware, and malware.
Features
•	Real-time News Integration: A news button on the homepage fetches live cybersecurity updates via an API.
•	Courses & Lessons: Four main courses covering:
1.	Evolution of Ransomware
2.	Phishing
3.	Scan Box
4.	Botnet
•	Quizzes: Each lesson ends with a quiz to test knowledge.
•	User Roles:
o	Normal Users: Can view and choose courses, take lessons, and complete quizzes.
o	Admin Users: Can upload new lessons, delete lessons, and view login history.
•	About Us Page: Information about the mission and team behind Cyber Shield.
•	Contact Page: Feedback form for users to share thoughts or issues.
•	Authentication: Sign up / log in functionality for both users and admins.
Tech Stack
•	Frontend: HTML5, HTML,CSS, JavaScript
•	Backend: JavaScript (Node.js)
•	API Integration: Real-time cybersecurity news feed

Usage
Visit the homepage to access real-time cybersecurity news.
Explore the Courses section to learn about phishing, ransomware, scan boxes, and botnets.
Complete quizzes at the end of each lesson to test your knowledge.
Admins can log in to manage lessons, upload or delete content, and monitor login history.
