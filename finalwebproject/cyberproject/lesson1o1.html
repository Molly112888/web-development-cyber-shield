How Ransomware Attacks Begin and Spread

Lesson Overview

Ransomware attacks begin quietly. Groups like GandCrab and REvil planned operations long before encryption using stolen access and hidden network movement.

1. Initial Access

  • Phishing emails
  • Stolen credentials
  • Unpatched systems
  • Exposed remote access
Key Concept: Many ransomware gangs buy access from Initial Access Brokers.

2. Privilege Escalation

  • Admin account creation
  • Security tool shutdown
  • Data theft

3. Lateral Movement

Attackers spread silently across internal systems.

4. Double Extortion

  • Pay to decrypt
  • Pay to prevent data leaks

5. Prevention

  • Regular patching
  • Multi‑factor authentication
  • Offline backups
